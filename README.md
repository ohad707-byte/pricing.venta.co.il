# Venta Auto Pricer V2

ממשק קטן לתמחור אוטומטי ראשוני מתוכניות PDF + קובץ תמחור קיים.

## הרצה
```bash
pip install -r requirements.txt
streamlit run app.py
```

## מה הממשק עושה
- העלאת תוכניות PDF ומחירון/תמחור Excel.
- חילוץ טקסט בסיסי מהתוכניות.
- זיהוי פריטים לפי דגמים/מילות מפתח: EMD-SQ, ELVOMV4PI, ELVID, ARNU, CO, GAS, CFM, BTU, מפוחים, תריסים ותעלות.
- זיהוי קומות טיפוסיות לפי שם קובץ, כולל 8-25 וכדומה.
- חישוב כמות משוערת, סימון ביטחון, והפקת Excel מסכם.

## חשוב
זו גרסה ראשונה אוטומטית-חוקית. היא לא מחליפה בדיקה ידנית, אבל נותנת בסיס מהיר לכתב כמויות ותמחור.
